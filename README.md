# Emily's Chatapp Submission
### A browser based chat application built on spring boot, MySQL and JS

# _Currently implemented:_
- Registration of new Users
- Login of registered Users
- Logging out of logged in User
- Connecting to main chat forum and chatting in said chat forum with simple Visuals
- Saving of Registered Users, Chats and Messages in a MySQL based Database
